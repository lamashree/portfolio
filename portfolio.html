Clicky Game

Clicky Game is the memory game created in React. This is a memory game assignment is built using React components, manage component state, and respond to user events. It renders different images which listen to click event from the user and keep track of which image user has been click. Base on the click event the game application has logic to reset the game, inform you select the right image, or select the image that already been clicked and keep track of your highest score.
By developing this clicky game I learned different types of reusable React Components we can use in our application, create React App, and how I can keep track of the score by maintaining the state of App.
